About me
I am studying Design Engineering at Imperial College with the hope to specialise in software or robotics. I pride myself in both my technical ability and my effective communication within a team, including across different engineering disciplines, to aid in the smooth running of projects. Furthermore, no engineering project is complete until it has been reported on and I enjoy writing and presenting the key findings at the end of an undertaking.

In my free time, I like to explore London, go on long runs and roll up sushi for my family and friends.

An Ode To Cardboard
The challenge was to design and build an electromechanical machine using cardboard and an Arduino. I decided to make a four-legged bug for children that used Klann linkages, DC motors, an IR sensor and an ultrasonic sensor (for collision prevention). Skills: Cardboard prototyping, electromechanical design.
Quiznconquer
A continuation of a university project in which we built a web app. This is a four player game that aims to make learning facts and vocabulary fun. The GitHub repository can be found at https://github.com/hannahjayneknight/quiznconquer. Skills: Debugging techniques, configuring a DNS, opening and using an AWS instance, JavaScript, HTML, SQLite, CSS, UI/ UX.
Pose
A team-based university project designed to improve desk health. Our final concept measures back posture using pressure sensors in a seat-pad. This measurement is sent to a desk companion via Bluetooth which mimics the user’s posture, providing a gentle nudge. Skills: Textile-based high-fi prototyping, video producing, conducting interviews with users, product design, user-based design, circuit design.
